# Archivos del conjunto de datos

Esta carpeta contiene el conjunto de datos utilizado para los experimentos de ML.

## Archivos

- `training_examples.json`: conjunto de datos inicial utilizado en esta iteración.
- `system_prompt.txt`: prompt base de instrucciones para generar artefactos de modelo.
- `processed/`: particiones de entrenamiento/validación/test y estadísticas del conjunto de datos generadas en el preprocesamiento.

## Origen del conjunto de datos

Los ejemplos se generan a partir de datos de landmarks de OpenStreetMap y se transforman en pares prompt/respuesta para LandmarkLens.

## Enlace externo del conjunto de datos

- OpenStreetMap: https://www.openstreetmap.org
