# Salidas de experimentos

Esta carpeta contiene artefactos de experimentación y cuadernos.

## Archivos

- `LandmarkLens_Examples.ipynb`: cuaderno de exploración.
- `evaluation_report.json`: métricas cuantitativas de `ML/scripts/evaluate_model.py`.
- `online_eval_report.json`: métricas de inferencia en línea de `ML/scripts/evaluate_online_ollama.py`.
- `export_eda_figures.py`: exporta figuras EDA en PNG desde el conjunto de datos crudo.
- `figures/`: recursos visuales exportados para informes (`eda_basic_overview.png`, `eda_advanced_cooccurrence_boxplot.png`, `eda_long_tail_coverage.png`).
